Some changes to my site need some more thought and testing before they go live.

Here's what's currently in progress:

## Cooks

"Cooks" are a new post type for things I've cooked.

I don't plan on publishing these often, so I am not that concerned about finishing it. 

It was interesting to think about posting "live" events on my website.

Most of the hard work is done by my [Wireless Thermometer](https://fireboard.com), which has an [API](https://fireboard.io) that lets me extract the data. It has a way of adding photos to its own notes section, but it doesn't have a way of getting them back out after, so this needs some more thought to make the posts more interesting.

[Preview](https://paultibbetts-uk-feat-cooks.preview.lab.paultibbetts.uk/cooks/2026-07-03-pulled-pork/)

## Name

Adds my name to the top of the page.

I didn't do this in the initial design because it's in the URL, and it seemed too obvious, but it does look a bit weird on posts when it just says "liked a post".
