I've seen a few indie web directories popping up lately, so in an attempt to keep track of them all I've started on a [directory of indie web directories](/indieweb-directories).

Then I was introduced to [Russel's paradox](https://en.wikipedia.org/wiki/Russell's_paradox), which asks if a directory should include itself?

The answer is **no**, you make [another one](/directory-of-indieweb-directories).
